# Hospital Management System (Java Swing)

This is a simple GUI-based Hospital Management System built using Java Swing.

## Features
- Add Doctor
- Add Patient
- View Doctors
- View Patients
- Clear Input Fields

## How to Run
1. Compile the program:
   javac HospitalGUI.java

2. Run the program:
   java HospitalGUI
